1. Cài những thư viện cần thiết bằng cmd:
	```
	pip install -r requirements.txt
	```
2. Sau đó cài browser cho playwright bằng lệnh dưới đây, tham khảo tại https://playwright.dev/python/docs/intro#installation
	```
	playwright install
	```
